TextScanner - приложение, улучшающее "читаемость" сфотографированных страниц с текстом, а также
предоставляющее набор инструментов для ручной корректировки процесса преобразования.

Использование:
В качестве аргументов передается название файла, который необходимо улучшить.
После запуска предлагается задать необходимые параметры изображения или же сразу вывести результат
с дефолтными параметрами. Для отображения результата, нажмите Enter. 
Чтобы выйти, нажмите 'q'. 
Чтобы сохранить результат, нажмите 's'

Графический интерфейс:
Графический интерфейс реализован по принципу MVC.
GraphController (контроллер) - отвечает за обработку действий пользователя;
GraphData (модель) - отвечает за доступ к данным, где в качестве данных хранятся исходное и 
измененное изображения, над которым осуществляются преобразования, и изменяемые параметры;
GraphGUI(представление) - модуль, осуществляющий отображение данных.

Модуль ThresholdMapCreator позволяет из исходного изображения получить преобразованную карту порогов.
Метод getThresholdMap() возвращает карту порогов для входного изображения, используя соответствующий тип
креатора (перечислены в creators). По умолчанию используется PyramidalCreator - алгоритм пирамидального 
разложения (см. статью https://habr.com/ru/company/abbyy/blog/218285/ - Повышение визуального качества 
для фотографий документов).
Так как карта порогов теоретически может быть получена различными способами, а не только с помощью
пирамидального разложения, мы оставим возможность расширения этого модуля (до полноценной фабрики),
путем расширения списка доступных креаторов. 

Используемый по умолчнию PyramidalCreator для получения карты порогов задействует модуль PyramidalDecompositor,
осуществляющий построение пирамиды по задаваемой стратегии Policy (например, пирамида минимумов, максимумов
или средних). 
Возвращает вектор подматриц (SubMat), соответствующих каждому уровню разложения.

Trackbar - дополнительная обертка над трекбарами для удобного взаимодействия контроллера и модели.
